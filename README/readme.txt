Projet C++ : Simulation d'un compte Bancaire ave le langage C++

Technologie utilisée : C++, sqlite3

Fonctionnement du programme : 

	On a une interface utilisateur en console qui se renseigne sur l'identité de l'utilisateur si l'utilisateur est une Banque avec vérification du mot de passe de la Banque alors la banque a acces a 
son interface console qui a les privilege d'ajouter, supprimer et lister un utilisateur ou plusieurs utilisateur.
Pour chaque utilisateur un numero de compte est generer de manier aleatoire composé de chiffre et de lettre permettant a un utilisateur de se connecter 
a son propre interface console.


Suite du projet : 
	Pour la suite du projet je voudrais ajouter une nouvelle technologie QT 
pour l'interface utilisateur, proposer une application web et mobile tout en garantissant la sécurité.

	Ajouter un espace de don pour les personnes en difficultes et/ou malades

Difficultés : 
	Manque de temps poour la finalisation du projet dans sa totalité et d'accompagnement dans la correction des erreurs lors du developpement
